# 快速开始指南

## 5分钟快速上手

### 第一步：安装依赖

打开命令行，进入项目目录：

```bash
cd excel-qa-system
pip install -r requirements.txt
```

### 第二步：创建测试数据（可选）

```bash
python test_data.py
```

这将在 `test_data` 目录下创建示例Excel文件，用于测试系统功能。

### 第三步：启动服务

```bash
python app.py
```

或者直接双击 `start.bat` 文件（Windows系统）。

### 第四步：访问系统

在浏览器中打开：http://localhost:5000

## 使用流程

### 1. 上传文件

- 拖拽Excel文件到上传区域
- 或点击"选择文件"按钮
- 支持 .xlsx, .xls, .csv 格式

### 2. 查看数据

上传成功后，系统会自动显示：
- 数据预览（前10行）
- 文件基本信息（行数、列数）
- 列信息（数据类型、空值统计）

### 3. 智能问答

在右侧对话框输入问题，例如：

**基础问题：**
- "数据有多少行？"
- "有哪些列？"
- "数据类型是什么？"

**统计问题：**
- "统计信息"
- "平均值是多少？"
- "总和是多少？"

**数据质量：**
- "哪些列有空值？"
- "有多少缺失数据？"

## 测试数据说明

系统提供了三个测试数据集：

### 1. 销售数据（test_data/销售数据.xlsx）
- 10行销售订单记录
- 包含：订单ID、产品名称、类别、单价、数量、销售额、客户姓名、地区、日期、状态
- 适合测试：销售统计、地区分析、产品对比

### 2. 员工信息（test_data/员工信息.xlsx）
- 8行员工记录
- 包含：员工ID、姓名、部门、职位、年龄、薪资、入职日期、工作年限、绩效评分
- 适合测试：人员统计、薪资分析、部门分布

### 3. 产品库存（test_data/产品库存.xlsx）
- 12行产品记录
- 包含：产品编号、产品名称、品牌、类别、库存数量、单价、库存价值、供应商、仓库位置
- 适合测试：库存管理、价值分析、品牌对比

## 常见问题

### Q: 如何使用OpenAI GPT进行智能问答？

1. 复制 `.env.example` 为 `.env`
2. 在 `.env` 文件中填入你的OpenAI API密钥：
   ```
   OPENAI_API_KEY=sk-your-api-key-here
   ```
3. 重启服务器

### Q: 没有OpenAI API密钥怎么办？

系统内置了规则匹配模式，可以回答常见的数据查询问题，无需API密钥。

### Q: 支持多大的文件？

默认支持最大16MB的文件，可以在 `app.py` 中修改 `MAX_CONTENT_LENGTH` 配置。

### Q: 数据会保存多久？

数据存储在服务器内存中，重启服务后会清空。如需持久化存储，可以扩展使用数据库。
