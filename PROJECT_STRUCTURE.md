## 核心文件说明

### app.py - 后端核心

**主要功能模块：**

1. **文件上传处理**
   - 支持拖拽和点击上传
   - 文件格式验证
   - 文件大小限制（16MB）

2. **数据解析**
   - Excel文件读取（.xlsx, .xls）
   - CSV文件读取
   - 数据类型识别

3. **数据分析**
   - 基本统计（行数、列数）
   - 数据类型分析
   - 空值统计
   - 数值列/文本列分类

4. **自然语言问答**
   - OpenAI GPT集成（可选）
   - 内置规则匹配
   - 上下文理解

5. **API接口**
   - RESTful API设计
   - CORS跨域支持
   - 错误处理

### index.html - 前端界面

**主要功能模块：**

1. **用户界面**
   - 响应式设计
   - Bootstrap 5样式
   - 现代化UI组件

2. **文件管理**
   - 上传区域（拖拽支持）
   - 文件列表展示
   - 文件选择和删除

3. **数据展示**
   - 表格预览
   - 分页浏览
   - 列信息展示

4. **问答交互**
   - 聊天界面
   - 消息展示
   - 建议问题

## 数据流程

```
用户上传Excel文件
    ↓
Flask接收并验证文件
    ↓
Pandas读取Excel数据
    ↓
分析数据结构（列名、类型、统计）
    ↓
存储到内存（uploaded_files字典）
    ↓
前端展示数据预览
    ↓
用户提问
    ↓
构建数据上下文
    ↓
调用问答引擎（OpenAI或规则匹配）
    ↓
返回答案给用户
```

## 配置说明

### 环境变量（.env）
```
OPENAI_API_KEY=your_api_key_here  # OpenAI API密钥（可选）
```

### 应用配置（app.py）
```python
UPLOAD_FOLDER = 'uploads'              # 上传目录
ALLOWED_EXTENSIONS = {'xlsx', 'xls', 'csv'}  # 允许的文件类型
MAX_CONTENT_LENGTH = 16 * 1024 * 1024  # 最大文件大小（16MB）
```

## 部署建议

### 开发环境
- 使用Flask内置服务器
- 启用debug模式
- 本地文件存储

- 禁用debug模式

- 配置数据库
